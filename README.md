# Vibe Coding – Justin Larsen


## How to run


### Part 1 (Vanilla JS)
1. Open `attempts/attempt1.html` in your browser.
2. Test playthroughs; try to break it (double‑clicks, out‑of‑turn clicks, etc.).
3. Open `attempts/attempt2.html` and compare changes.
4. Use `tictactoe-final.html` for submission.


### Part 2 (SPA)
1. Open `spa-version/spa.html` in your browser.
2. If your browser blocks module/JSX execution from file URLs, right‑click → Open With → a different browser **or** start a tiny local server, e.g. `python -m http.server` and visit `http://localhost:8000/spa-version/spa.html`.


## What’s included
- Prompts you can paste as‑is into your AI tool (see `PROMPTS.md`).
- Reflection template with sentence starters (see `REFLECTION.md`).

1. Create folder `vibe-coding-justin-larsen/` in your repo.
2. Place the files as shown in this canvas.
